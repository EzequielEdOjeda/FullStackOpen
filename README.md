# [Full Stack Open](https://fullstackopen.com/es/)

Curso del Departamento de Ciencias de la Computación de la Universidad de Helsinki y Houston Inc.

Sirve como introducción al desarrollo moderno de aplicaciones web con JavaScript. El enfoque principal es la creación de aplicaciones de una sola página con ReactJS que utilizan API REST desarrolladas con Node.js.

Este curso también abarca GraphQL, una alternativa moderna a las API REST. Además, abarca las pruebas, la configuración y la gestión del entorno, y el uso de MongoDB para almacenar los datos de la aplicación. La edición de este año también incluye una 13 Partes. Este repositorio contiene mis soluciones para los ejercicios de este curso.
